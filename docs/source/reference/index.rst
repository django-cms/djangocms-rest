Reference
=========

Exact, endpoint-by-endpoint documentation: paths, parameters, response fields and example
payloads. For task-oriented instructions see the :doc:`../how-to/index`; for the ideas
behind the API see the :doc:`../explanation/index`.

.. note::

   These pages describe the API by hand. When ``drf-spectacular`` is installed you also
   get a **live, always-current** OpenAPI schema and browsable docs generated from the
   running code — see :doc:`../how-to/generate-openapi-schema`. Treat the live schema as
   authoritative for your installed version and these pages as the explained companion.

Start with :doc:`conventions` — the cross-cutting rules (language prefix, ``preview``,
``html``, ``X-Site-ID``, pagination, errors) that apply across endpoints and are not
repeated in full on every page.

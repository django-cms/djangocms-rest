How-to Guides
=============

Each guide solves one concrete problem. They assume you already have djangocms-rest
installed and working (see the :doc:`../tutorial/index`) and that you are comfortable with
django settings and URLs.

If you want to understand the concepts behind a guide rather than accomplish a task, see
the :doc:`../explanation/index` instead.

Contribute a guide
------------------

Solved a scenario that isn't covered here — apphook routing, image handling, a framework
integration? Please share it. Open a pull request or an issue on the
`GitHub repository <https://github.com/django-cms/djangocms-rest>`_.

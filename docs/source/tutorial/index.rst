Tutorial
========

This is the learning path for djangocms-rest. It is a single, continuous lesson: by the
end you will have installed the package into a django CMS project and fetched real page
content over the REST API.

The tutorial deliberately sticks to the default, published-content setup. Optional
features — CORS, multiple sites, draft previews, the OpenAPI schema, custom plugin
serializers — are each covered as a focused recipe in the :doc:`../how-to/index`.
